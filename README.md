# Drawing Math Fuctions

This project is made to show how to draw math functions in Unity and applying shaders to them to look cool.
I started by following Jasper Flicker's [tutorials](https://catlikecoding.com/unity/tutorials/basics/) and adapted to the newest Unity version, currently version 6.x.x.
